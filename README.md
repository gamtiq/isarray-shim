# isarray-shim

Shim for ECMAScript 5 Array.isArray

## Installation

Install component:

    npm install -g component

Then:

    component install gamtiq/isarray-shim

## Usage

    var isArray = require("isarray-shim");
    ...
    var test1 = Array.isArray(someObj1);
    var test2 = isArray(someObj2);
